# Rain Dance

__Is this ...rain currently in the room with us?__


This repo attempts to answer a very simple question: Will it (really) rain today?
For too long, I have trusted news sites to report the weather to me truthfully.
But the data is too overwhelming to argue against... they're not.

We'll do some analysis on meteorological data and compare it against one
specific news website's claims of rain occurrence where I currently live.

Go to the associated github.io page [here](https://temataro.github.io/rain-dance/)

Get Quarto [here](https://quarto.org/docs/get-started/) and [uv](https://docs.astral.sh/uv/getting-started/installation/) to render or rebuild this.
